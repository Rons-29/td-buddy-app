## 🚀 実装中

### Phase 4: 実用機能拡張

#### **🔤 Step 13: 文字・テキスト系ツール実装** ← 現在実装中
**目標**: QA業務で頻繁に使用される基本的な文字・テキスト処理機能を提供

**実装機能**:
1. **📝 テキスト生成ツール**
   - ひらがな・カタカナ・漢字・英数字・記号の組み合わせ
   - 指定文字数でのランダムテキスト生成
   - 文字種別指定機能

2. **🔄 旧字体変換ツール**
   - 新字体→旧字体変換機能
   - 変換辞書の実装
   - 旧字体テキストのランダム生成

3. **📊 文字数カウントツール**
   - 文字数・バイト数・行数カウント
   - 全角・半角・英数字の分類カウント
   - リアルタイム文字数表示

4. **🔍 文字種分析ツール**
   - 含まれる文字種の詳細分析
   - 文字コード判定
   - 特殊文字・絵文字の検出

**UI/UX設計**:
- `/text-tools` ページの新設
- タブ切り替えによる機能分離
- リアルタイムプレビュー機能
- コピー・ダウンロード機能

**技術実装**:
- フロントエンド: Remix + TypeScript
- バックエンド: NestJS API
- 文字処理ライブラリの導入
- 旧字体変換辞書の構築
